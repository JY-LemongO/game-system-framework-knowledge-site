Build Plan

Implementation Roadmap

로드맵은 게임 시스템 프레임워크를 한 번에 모두 만들지 않고, 테스트 가능한 단위로 쌓아 올리는 순서를 제안한다. Phase 2 기준으로 Combat Resolution과 Status System까지 본문 구조가 완성되었다.

MVPTestabilityDebugTraceVertical Slice

권장 구현 순서

Phase 0. Core Runtime Contract

EntityId, SourceId, TagSet, EventBus, TimeService, RandomStream, DebugTrace를 먼저 정의한다.

Phase 1. Stat System MVP

StatDefinition, StatInstance, Modifier, 계산 순서, ResourceStatController, Debug Breakdown을 구현한다.

Phase 2. Effect System MVP

EffectDefinition, EffectContext, TargetSelector, ValueExpression, EffectOperation, EffectResult를 구현한다.

Phase 3. Skill / Action MVP

SkillDefinition, SkillRuntime, SkillValidator, CostPolicy, CooldownPolicy, ActionTimeline, EffectBundle 실행을 구현한다.

Phase 4. Combat Resolution MVP

DamageRequest, HitResolver, DamageFormula, Defense/Resistance/Shield 단계, DamageResult와 CombatLog를 구현한다.

Phase 5. Status System MVP

StatusDefinition, StatusInstance, StackPolicy, DurationPolicy, TickPolicy, ModifierProvider, DispelPolicy를 구현한다.

Phase 6. Fireball Vertical Slice

Fireball 하나로 Skill → Effect → Combat → Status → EventLog → DebugTrace 전체 흐름을 통합 테스트한다.

Phase 7. Equipment / Item System

ItemDefinition, ItemInstance, EquipmentSlot, Affix, SetBonus, TriggeredEffect를 추가한다.

Phase 8. Progression / Reward System

LevelTable, RewardDefinition, TalentNode, SkillMutation, RewardApplicator를 추가한다.

Phase 4 Combat 구현 체크리스트

  • DamageRequest와 DamageResult 타입을 먼저 고정한다.
  • Hit, Critical, Defense, Resistance, Shield 단계를 함수 단위로 분리한다.
  • 피해 계산 순서와 반올림 정책을 CombatFormulaDefinition에 기록한다.
  • DamageResult에 계산 로그, 흡수 피해, 실패 이유를 포함한다.
  • DamageEffect가 직접 피해 공식을 갖지 않도록 CombatResolver에 위임한다.

Phase 5 Status 구현 체크리스트

  • StatusDefinition과 StatusInstance를 분리한다.
  • StackPolicy와 DurationPolicy를 데이터로 표현한다.
  • 상태 적용 원천은 appliedBySourceId로 남기고, StatusModifier의 sourceId는 statusInstanceId로 설정한다.
  • Status 제거 시 Modifier, tick 예약, 이벤트 구독을 함께 정리한다.
  • Burn 같은 DoT는 periodic DamageEffect로 처리한다.

Fireball 기본 명중마나 차감, 쿨다운 시작, DamageResult 생성, HP 감소
Fireball + 보호막shieldAbsorbed 기록과 최종 HP 피해 확인
